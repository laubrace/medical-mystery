# Clarity Care

A simple, step-by-step tool to help people organize complex medical histories and get clearer next-step guidance.

## Goal
Make it easier for people to:
- organize symptoms and medical history
- identify meaningful patterns
- prepare for doctor visits
- feel supported when answers are unclear

## Core Experience (v1)
- single intro screen
- guided step-by-step intake
- optional document upload
- clarifying questions
- simple results output
- optional premium screen

## Design Principles
- simple > overwhelming
- minimal typing required
- supportive, not clinical
- low-energy friendly
- can stop and return anytime

## Tech (v1)
- HTML
- CSS
- Vanilla JavaScript

## Status
